## Validación del Modelo
Se realizaron pruebas de consistencia utilizando contenido del archivo `reviews.csv`, validando:
* **Casos Positivos:** Identificación correcta de sentimientos favorables.
* **Casos Negativos:** Detección de críticas y feedback negativo.
* **Integración:** El motor de inferencia (`engine.py`) procesa correctamente las entradas externas antes de su exposición vía API.
